## General info
When you enter website it detects your ip address and shows your approximate location

## Technologies
Project is created with:
### Front-End
* leaf 1.0.1
* leaflet 1.7.1
* leaflet-draw 1.0.4
* parcel-bundler 1.12.5
* bootstrap 5.1.3
* bootstrap-icons 1.8.1
* SCSS/SASS


## Setup
To run this project:
* ```$npm i -D parcel-bundler```
* ```$npm run dev```

To create a build:
* ```$npm run build```
